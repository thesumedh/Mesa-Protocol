# Security Policy

## Supported Versions

Only the latest release version of Mesa Protocol is supported for security updates.

| Version | Supported |
| ------- | --------- |
| v0.2.x  | Yes       |
| < v0.2  | No        |

## Reporting a Vulnerability

We take the security of Mesa seriously. If you find a security vulnerability, please do not report it publicly (via GitHub Issues or discussions). Instead, please report it directly by emailing:

**security@mesa.example.org**

Please include the following details in your report:
- A description of the vulnerability and its potential impact.
- Step-by-step instructions (with code if possible) to reproduce the issue.
- Any suggested mitigations or fixes.

We will acknowledge receipt of your report within 48 hours and work to provide a patch in a timely manner.
