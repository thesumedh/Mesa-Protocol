# Mesa — Implementation Roadmap

This roadmap tracks the development phases of Mesa as a self-hosted embedded finance runtime for Stellar.

---

## 🗺️ Phase-wise Development

| Phase | Milestone | Status | Focus Areas |
|:---|:---|:---|:---|
| **Phase 1** | **Milestone 1: Runtime Foundation** | **✅ Completed** | Durable execution scheduler, executor, Postgres store, and event-sourcing pipelines. |
| **Phase 1** | **Milestone 2: SDK & Dev Experience** | **✅ Completed** | Fluent `Mesa.flow()` builder with design-time parameter and format checks. |
| **Phase 1** | **Milestone 3: Provider Framework** | **✅ Completed** | Standardized `MesaProvider` with health check. Initial `AnchorProvider` (SEP-24). |
| **Phase 1** | **Milestone 4: Workflow Dashboard** | **✅ Completed** | HTML developer console showing executions, steps, events, and webhook resume actions. |
| **Phase 1** | **Milestone 5: Real Stellar Integration** | **✅ Completed** | Real Horizon integration, SEP-10 challenge transaction signing, and live Testnet corridor. |
| **Phase 2** | **Milestone 6: Developer Polish** | ⬜ **In Progress** | Production-ready README, Quickstart guide, developer onboarding. |
| **Phase 2** | **Milestone 7: Demo Polish** | ⬜ **Planned** | Clean presentation setup, visual logs validation guide. |
| **Phase 2** | **Milestone 8: Validation** | ⬜ **Planned** | Collect feedback from Stellar developers, mentors, and Build Station teams. |
