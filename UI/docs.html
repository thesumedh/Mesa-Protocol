Mesa Protocol SDK

Mesa is an open-source workflow orchestration engine built specifically for Stellar fintech applications. It sits between your application logic and the Stellar network, handling multi-step transactions, retries, SEP-10 anchors, and Soroban smart contract calls with durable state safety.

replay

Durable Retries

Ledger congestion or Horizon RPC timeouts trigger automatic exponential backoff retry loops.

save

Postgres Persistence

Checkpoints state transitions to database storage so workflow progress is preserved through restarts.

security

HMAC Webhooks

Webhook callbacks use HMAC-SHA256 signature validation for secure asynchronous state updates.

Docs chevron_right Installation

Quick Installation

Install the Mesa SDK package into your Node.js or Next.js application using npm, yarn, or pnpm.

terminal
npm install @mesaprotocol/sdk

Basic Initialization

import { Mesa } from '@mesaprotocol/sdk';

// Configure Mesa runtime endpoint
Mesa.configure({
  runtimeUrl: 'http://localhost:3001',
  apiKey: process.env.MESA_API_KEY
});
Docs chevron_right SDK API Reference

Fluent FlowBuilder API

Mesa SDK uses a fluent method-chaining builder to define workflows safely.

.receive(params) Deposit Listener

Suspends flow until an incoming payment matching asset and minimum amount is received.

.receive({ asset: 'XLM', minAmount: 10, toAddress: 'G...' })
.sep10Auth(params) Anchor Authentication

Executes Stellar SEP-10 challenge signing and acquires anchor JWT token.

.sep10Auth({ anchorUrl: 'https://anchor.stellar.org' })
.convert(params) Asset Swap

Swaps assets on Stellar DEX or through an anchor rate query.

.convert({ fromAsset: 'XLM', toAsset: 'USDC' })
.payment(params) Stellar Transfer

Submits a payment operation to Horizon network with automatic sequence management.

.payment({ toAddress: 'GA4U...', amount: 25, horizonUrl: 'https://horizon-testnet.stellar.org' })
Docs chevron_right Stellar Protocols

Native Stellar Integrations

Mesa natively supports Horizon RPC, Soroban smart contracts, SEP-10, SEP-24, and secret reference tokens. 

import { Mesa } from '@mesaprotocol/sdk';

const flow = Mesa.flow('stellar-remittance-pipeline')
  .receive({ asset: 'XLM', minAmount: 25 })
  .sep10Auth({ anchorUrl: 'https://anchor.stellar.org' })
  .convert({ fromAsset: 'XLM', toAsset: 'USDC' })
  .payment({ toAddress: 'G...', amount: 25 })
  .compensate({ reason: 'Refund user on execution error' })
  .build();

await Mesa.execute(flow);
